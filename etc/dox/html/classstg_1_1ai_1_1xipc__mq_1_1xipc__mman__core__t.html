constexpr static auto stg::ai::xipc_mq::xipc_mman_core_t::null_seq { SIZE_MAX }
staticconstexprprotected

NULL sequence value

We need to avoid branch prediction and want to describe 'empty' storage. When container is empty, all sequences equals to null sequences. But initial placement is free from branches, because SIZE_MAX + 1 = 0, i.e. index in the chunks virtual array
